# Smart Creative Studio Booking Dashboard

## Deskripsi Singkat

Proyek ini adalah website modern dan responsif untuk Smart Creative Studio yang menyediakan pengalaman booking studio secara praktis dan menarik. Website ini terdiri dari tiga bagian utama, yaitu landing page, dashboard, dan katalog studio, serta dilengkapi dengan form booking dalam bentuk popup untuk mempermudah pengguna melakukan reservasi.

## Tujuan Proyek

Tujuan utama dari proyek ini adalah:

- Menyediakan tampilan website yang profesional dan menarik untuk studio kreatif.
- Memudahkan pengguna melihat informasi studio, ketersediaan, dan status booking.
- Menghadirkan pengalaman reservasi yang sederhana, cepat, dan intuitif.
- Menyajikan dashboard ringkasan data booking secara visual.

## Fitur Utama

- Landing page yang informatif dan estetik.
- Halaman dashboard untuk melihat ringkasan pemakaian studio.
- Halaman katalog studio untuk melihat daftar studio yang tersedia.
- Form booking dalam popup modal.
- Penyimpanan data booking menggunakan localStorage agar data tetap tersimpan di browser.
- Tampilan responsif untuk desktop maupun perangkat mobile.

## Struktur Project

- index.html: halaman utama (landing page).
- views/component/dashboard.html: halaman dashboard.
- views/component/catalog.html: halaman katalog studio.
- views/style/style.css: file stylesheet utama untuk desain website.
- views/script/script.js: file JavaScript untuk interaksi, validasi form, dan penyimpanan data.

## Teknologi yang Digunakan

- HTML untuk struktur halaman.
- CSS untuk desain, layout, dan responsivitas.
- JavaScript murni untuk interaksi pengguna dan logika aplikasi.
- LocalStorage untuk menyimpan data booking secara lokal.

## Penjelasan Teknis

Website ini dibangun dengan pendekatan sederhana namun fungsional. Struktur halaman dipisahkan berdasarkan tujuan penggunaannya, sehingga landing page, dashboard, dan katalog studio berada pada halaman yang berbeda. Penggunaan JavaScript murni memungkinkan fitur interaktif seperti popup booking, validasi input form, update data dashboard, dan penyimpanan data booking tanpa memerlukan framework tambahan.

### Komponen Utama

1. Landing Page
   - Menampilkan gambaran umum studio.
   - Memberikan akses cepat ke katalog studio dan dashboard.

2. Dashboard
   - Menampilkan informasi penting seperti jumlah booking, kapasitas, dan status studio.
   - Menyajikan data ringkasan secara visual untuk memudahkan pemantauan.

3. Catalog Page
   - Menampilkan daftar studio dalam format kartu.
   - Pengguna dapat memilih studio dan mengisi form booking melalui popup.

4. Booking Modal
   - Form booking muncul pada popup modal.
   - Menggunakan validasi input sebelum data disimpan.

Pengguna tidak perlu memahami teknologi untuk dapat menggunakan website ini karena alur yang disediakan sederhana dan intuitif.

- Belum terintegrasi dengan database backend.
- Fitur notifikasi dan autentikasi pengguna masih dapat dikembangkan lebih lanjut.
